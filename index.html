Konfigurasi API
Hubungkan ke Google Sheets
1
Buat Google Sheet & Jalankan initSheets()

1. Buka sheets.google.com → buat spreadsheet baru

2. Klik Extensions → Apps Script

3. Paste seluruh isi file apps-script.js ke editor

4. Klik tombol ▶ Run → pilih fungsi initSheets → klik Run

5. Izinkan akses jika diminta

2
Deploy sebagai Web App

1. Di Apps Script → klik Deploy → New deployment

2. Klik ⚙️ → pilih Web App

3. Description: SiPaket API v1

4. Execute as: Me

5. Who has access: Anyone

6. Klik Deploy → copy URL yang muncul
